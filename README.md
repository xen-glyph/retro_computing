# Retro Computing

> *"Old machines don't belong on a shelf. They belong at work."*

Welcome to my retro computing workshop.

This repository contains software, utilities, documentation, and hardware projects dedicated to giving classic computers and peripherals meaningful jobs in the modern world.

While many people collect vintage hardware for nostalgia, my goal is a little different. I enjoy preserving these machines by making them useful again. Whether that's printing a daily weather fax on a 35+ year old dot matrix printer, connecting a 1970s thermal terminal to modern Linux systems, or networking CP/M computers that were never designed for today's world, I believe old hardware still has plenty left to give.

Modern computers are incredibly powerful.

Vintage computers are incredibly satisfying. No pop-up ads. No cookies. No clickbait.

This repository exists where those two worlds meet.

---

# Philosophy

Technology doesn't have to be disposable.

A machine that has faithfully served for forty years deserves more than a place in a museum.

If a Raspberry Pi can quietly bridge the gap between the Internet and a dot matrix printer...
If a Kaypro can once again exchange files...
If a dial-up modem can create a useful connection...

...then those machines are alive again.

That's the purpose of this repository.

# Current Workshop

## Vintage Hardware

### Grandpa
**1983 Kaypro 10**
*CP/M Portable Computer*

The old man of the shop. Grandpa is my primary CP/M machine and continues to prove that eight-bit computers still have useful work to do.

---

### Gordon
**1982 Kaypro II**
*CP/M Portable Computer*

Grandpa's older brother. Gordon represents one of the earliest generations of portable business computers and serves as both a restoration project and working CP/M system.

---

### Scribe
**1975 Texas Instruments Silent 745**
*Thermal Data Terminal with Acoustic Coupler*

Originally designed decades before the modern Internet, Scribe now serves as a network-connected hardcopy terminal. A reminder that "online" looked very different before Ethernet.

---

### Sensei
**OKI Microline 321 Turbo**
*9-Pin Dot Matrix Printer*

The workhorse.

Every morning Sensei automatically prints a one-page NOAA Weather Fax generated by Seraph. Loud, dependable, and still producing excellent output decades after leaving the factory.

# Modern Infrastructure

## Logos
**ASUS VivoBook**
*Linux Mint 22.3 Xfce*

Daily workstation.

Software development, scripting, documentation, and project coordination all begin here.

---

## Seraph
**Raspberry Pi 4 (4 GB)**

The bridge between generations.

Seraph interfaces modern services with vintage hardware, hosts utilities, manages scheduled jobs, stores archives, and quietly prepares information for devices that predate the Internet by years.

---

## Glyph
**Raspberry Pi Zero 2 W**

Portable cyberdeck and field companion.

Designed for lightweight networking, diagnostics, radio projects, and remote operations wherever they happen to be needed.

# Design Goals

Projects in this repository generally follow a few principles:

- Make vintage hardware genuinely useful.
- Favor simple, dependable solutions.
- Preserve the character of the original equipment.
- Bridge generations instead of replacing them.
- Build tools that I actually use every day.

---

# Why?

Because hearing a dot matrix printer come alive every morning to hand me the day's weather is simply more satisfying than checking another phone app.

Because a forty-year-old computer still deserves something interesting to do.

Because old hardware isn't obsolete when it still solves real problems.

And because keeping these machines working is one of the best ways to preserve computing history.
